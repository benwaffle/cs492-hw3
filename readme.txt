Instructions:

To compile, type `make`.
To run the code, type:
`./main -f <file_list.txt> -d <dir_list.txt> -s <disk size> -b <block size>`

File list should be generated by:
find . -type f -ls > file_list.txt

Dir list should be generated by:
find . -type d > file_list.txt

The available commands are:
- cd <directory>
- cd..
- ls
- mkdir <name>
- create <name>
- append <name> <bytes>
- remove <name> <bytes>
- delete <name>
- exit
- dir
- prfiles
- prdisk
